# 02-赚钱机制

## 1. 核心赚钱路径

```
买入（低价层） → 持仓等待 → 卖出（高价层） → 网格利润
```

## 2. 利润来源拆解

### 2.1 理想情况（28 次，58%）
- 在 Level 2/3 买入 @ 1975/2012.5
- 价格反弹至 Level 3/4 卖出 @ 2012.5/2050
- 单次利润：约 1.15 USDT / 0.03 ETH
- 持仓时间：几分钟到几小时

### 2.2 现实情况
- 不是所有买入都能盈利卖出
- 12 次被迫在更低价格卖出（亏损）
- 8 次同价位进出（库存再平衡）
- 4 次启动库存释放（无前序买入）

## 3. 关键数字

| 指标 | 数值 | 含义 |
|------|------|------|
| 网格利润率 | ~1.86% | 每格理论利润空间 |
| 实际盈利往返 | 28 次 | 真正赚钱的次数 |
| 实际亏损往返 | 12 次 | 被迫止损的次数 |
| 盈亏比 | 2.3:1 | 盈利次数 / 亏损次数 |
| 平均持仓 |  varies | 几分钟到 24 小时不等 |

## 4. 永续合约迁移要点

### 需要调整的
- **标的**：ETH-USDT → ETH-USDT-SWAP
- **杠杆**：现货 1x → 永续低杠杆（建议 2-3x）
- **保证金**：逐仓模式，控制单笔风险
- **资金费率**：长期持仓需考虑资金成本

### 保持不变的
- 网格逻辑：8 格等差，37.5 USDT 间距
- 价格区间：1900-2200（需根据市场调整）
- 核心原则：震荡市赚钱，趋势市保命

## 5. 风控底线

1. **不追涨**：价格突破 2200 不上移区间追涨
2. **不扛单**：单笔亏损超过 5% 强制平仓
3. **低杠杆**：逐仓 2-3x，避免爆仓
4. **及时止盈**：有利润就逐步兑现，不贪心
